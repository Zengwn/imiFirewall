The imiFirewall
================

The imiFirewall , based on the imiFirewall project, there are more modules added into this application. With the help of this application, you can filter the boring message and unknow call, also you can manage the permissions and installation of the software, even you can control the access of the application to the network installed on your devices.  However, all the thoese require the root privilege.
